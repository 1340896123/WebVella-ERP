WebVella ERP 
======
**WebVella ERP** 是一个免费、开源的 Web 软件，旨在极致定制和可插入性，以满足任何业务数据管理需求。它建立在我们的经验、最佳实践和最新的可用技术之上。目前它定位于使用 ASP.NET Core 7，我们选择的数据库是 PostgreSQL 11。可以在 Linux 或 Windows 操作系统上运行。目前只在 Windows 上进行了测试。
