WebVella企业资源规划
======
**WebVella ERP**是一款免费开源的网络软件，旨在为任何业务数据管理需求提供极端的定制和可插拔性服务。它建立在我们的经验、最佳实践和最新可用技术的基础上。目前它的目标是ASP.NET Core 7。我们选择的数据库是PostgreSQL 11。将Linux或Windows作为主机操作系统。目前仅在Windows上测试。
